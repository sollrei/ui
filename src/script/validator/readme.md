* @description 邮箱规则
* 1.邮箱以a-z、A-Z、0-9开头，最小长度为1.
* 2.如果左侧部分包含-、_、.则这些特殊符号的前面必须包一位数字或字母。
* 3.@符号是必填项
* 4.右则部分可分为两部分，第一部分为邮件提供商域名地址，第二部分为域名后缀，现已知的最短为2位。最长的为6为。
* 5.邮件提供商域可以包含特殊字符-、_、.
